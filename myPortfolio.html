My Work


MKChat


MKChat is chat web-app built with Nodejs, socket.io, bootstrap and CSS3. You can create a chatroom then send the link or the chatroom's unique ID to your friends to begin chatting! No registration required. The messages are deleted when you close your browser windows and the app is hosted on Heroku which offers SSL encryption for extra security.


You can find it here.


TvGuide


TvGuide is a Windows 8.1 Store app built with C#. The app will automatically pull from the internet all the latest Greek TV program info and display it to you. It can update itself on the background and display what's on your favorite channels right now on your Start screen with its Live Tile.


You can find it on the Windows Store.


Status Generator


Status Generator is a Windows 8.1 Store app build with C#. With this app you don't need to search the web any longer becuase you have 2000+ statuses right on your computer. Choose one of the categories to find the one you want or use the Random button. You can click the share button then to share the status with your friens on social media.


You can find it onn the Windows Store.


Information Health System


IHS is a web-app made using mixItUp library, php and sql for the backend. It is a full information health system for scheduling and managing medical examinations, patients and overall progress.


You can find the code on my Github.
